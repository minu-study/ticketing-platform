[[overview-http-verbs]]
== HTTP Methods

|===
| Method | Usage

| `GET`
| 리소스 조회

| `POST`
| 리소스 생성

| `PUT`
| 리소스 수정

| `DELETE`
| 리소스 삭제
|===

[[overview-http-status-codes]]
== HTTP Status Codes

|===
| Status code | Usage

| `200 OK`
| 요청 성공

| `201 Created`
| 리소스 생성 성공

| `400 Bad Request`
| 잘못된 요청

| `404 Not Found`
| 리소스를 찾을 수 없음

| `500 Internal Server Error`
| 서버 에러
|===
